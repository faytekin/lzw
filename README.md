# faytekin/lzw

## System Requirements

- PHP 7.2
- ext-mbstring - for handling multibyte string (should be available on most systems)

## Installation

You can install this package via composer:
``` bash
composer require faytekin/lzw
```

Or add the following to your composer file:

    {
        "require": {
            "faytekin/lzw": "^1.0"
        }
    }
